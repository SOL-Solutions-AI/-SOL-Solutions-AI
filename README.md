# SOLutions – Prep for VA SOL

SOLutions is a free, browser-based study tool designed to support K–12 students in Virginia with Standards of Learning (SOL) preparation. Powered by Playlab, it provides structured review, mastery exercises, and practice tests through an interactive embedded study bot.

## Features

- **Interactive SOL Study Bot**: Embedded directly in the webpage for immediate use.
- **Structured Workflow**: Setup → Diagnostics → Review → Mastery → Practice.
- **Student-Focused Design**: Clear instructions, safe responses, no personal data collection.
- **Adaptive Learning**: Supports notes, flashcards, mnemonics, and full-length practice tests.
- **Inclusive and Accessible**: Designed for a wide range of learning needs and classroom contexts.

## File Structure

- `index.html` – Main webpage with description and embedded SOL Study Bot.
- Optional assets (CSS, images) can be added in subfolders for further customization.

## How to Use

1. Open `index.html` in any modern browser.
2. Read the description to understand the workflow and functionality.
3. Scroll down to the **“Access the SOL Study Bot Below”** section.
4. Use the embedded bot to begin review, mastery, and practice exercises.

## Notes

- The embedded Playlab bot requires an active internet connection.
- No account or login is required to access the study bot.
